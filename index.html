Clima Precision

Aproximações feitas utilizando ferramentas de cálculo numérico através de análise de dados pluviométricos de Natal/Rio Grande do Norte

Métodos Numéricos

Para nossas análises, utilizamos o Método dos Mínimos Quadrados junto com a Série de fourier, ambos os métodos fornecem insights valiosos sobre o comportamento dos dados de precipitação de forma que podemos fazer um previsão sobre as chuvas da cidade.

MMQ

O Método dos Mínimos Quadrados, é uma técnica de otimização matemática que procura encontrar o melhor ajuste para um conjunto de dados.

Série de Fourier

Ferramenta que permite representar um sinal periódico como uma soma infinita de componentes senoidais.

Média de precipitação dos últimos 11 anos.

Foi elaborado um gráfico utilizando a média dos dados pluviométricos de cada ano, com o objetivo de proporcionar uma visão abrangente sobre o comportamento das chuvas nos últimos 11 anos.

Aproximação pelo método MMQ.

Utilizando o método dos Mínimos Multiplos Quadrados, aproximamos os dados pluviométricos coletados para obter uma função periódica dos períodos chuvosos na cidade de Natal.

  • Obtemos um gráfico de tempo em função dos dados pluviométricos.
  • Função de ajuste dos dados.
  • Aproximação por Fourier e MMQ.
